Data-Viewer

Web component

<data-viewer> is a web component for viewing table data:

  • Vanilla JS (no dependencies)
  • Reads json from url
  • Supports multi-index data structures

On this page you will find several examples of what you can do with this web component.

Specifications

  • Attributes
    • src
    • locale
    • na-rep (default: "-")
    • borders:
      • hide-group-borders
      • hide-row-borders
      • hide-thead-border
      • hide-index-border
  • Custom properties
    • --hover-color
    • --cursor (default: auto)
    • --border-color (default: currentColor)
    • --axes-width (default: 2px)
  • Events
    • data-changed
    • cell-click
